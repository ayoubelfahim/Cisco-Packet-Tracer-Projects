# Network Basics

## Overview

This project demonstrates a simple Local Area Network (LAN) using two computers connected to a Cisco switch.

## Objectives

- Configure IPv4 addresses
- Connect two PCs through a switch
- Verify connectivity using the ping command

## Network Topology

PC1 ---- Switch ---- PC2

## IP Configuration

| Device | IP Address | Subnet Mask |
|---------|------------|-------------|
| PC1 | 192.168.1.10 | 255.255.255.0 |
| PC2 | 192.168.1.20 | 255.255.255.0 |

## Verification

- Ping from PC1 to PC2
- Successful communication between both devices

- Cisco Packet Tracer
